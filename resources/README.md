# Resources

This folder contains auxiliary resources used by the Wasabi project logos, diagrams, test certificates, helper scripts, and anything else
that supports the repository but is not part of the core module or
documentation.

## Notes

- The files in this folder are **not required** to use Wasabi. They are
  here for development, documentation, and testing purposes.
- Test certificates under `certs/` are self‑signed and expire quickly;
  they are intended **only** for local TLS testing with WSS.
- Diagram sources (if any) live here so that documentation can reference
  them without depending on external hosts.
